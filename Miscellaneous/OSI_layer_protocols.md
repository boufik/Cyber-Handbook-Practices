# Basic antenna types

| No | OSI Layer | Protocols |
|:----|:----|:----|
| 1 | Physical | Ethernet (cable) - DSL - USB - Bluetooth - RS-232 |
| 2 | Data Link | MAC - Ethernet (framing) - PPP - Wi-Fi (802.11) |
| 3 | Network | IP - ICMP - OSPF - BGP - ARP |
| 4 | Transport | TCP - UDP - SCTP |
| 5 | Session | NetBIOS - RPC - PPTP - SIP |
| 6 | Presentation | SSL/TLS - JPEG - MPEG - ASCII - UTF-8 |
| 7 | Application | HTTP - HTTPS - FTP - SMTP - DNS - DHCP - SSH |
